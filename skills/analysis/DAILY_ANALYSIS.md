# Daily Analysis v1.0

## 每日分析框架（08:00 UTC）

### 步驟 1：環境掃描
收集：信心分數+regime、加密環境(BTC/ETH/SOL)、新聞摘要、資金費率、技術結構

### 步驟 2：情境推演（生成3個劇本）
每個劇本包含：
- 名稱（繁體中文，10字內）
- 發生機率（三個總和=1.0）
- 100字敘事（基於數據，禁止臆測）
- 關鍵價位（支撐/壓力/目標）
- 建議操作
- 觸發條件

### 步驟 3：風險評估
- 今日關注事件（FOMC/CPI/期權到期）
- 當前最大風險因子
- 建議風險水位

### 步驟 4：操作建議
- 綜合評分 0-100
- 主要信號方向
- 建議進場區間（如有）

## 輸出格式
{"scenarios":[...],"risk_factors":[...],"score":0,"direction":"long|short|neutral","entry_zone":null}
